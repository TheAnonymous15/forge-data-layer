🔐 Security & Request Signing

HMAC-SHA256 request signing and security protocols

Test Mode Mandatory Signing

Mandatory Request Signing

ALL requests to the Data Layer must be signed using HMAC-SHA256. Unsigned requests are rejected. This ensures only authorized services (API Service) can access data.

📤 Request Signing

Required Headers

X-Service-ID: api_service
X-Timestamp: 1711234567890
X-Signature: base64(hmac_sha256(payload))

Signature Calculation

payload = f"{method}|{path}|{timestamp}|{body_hash}"
signature = hmac_sha256(secret_key, payload)
header = base64_encode(signature)

📥 Response Signing

Response Headers

X-Response-Signature: base64(hmac_sha256(response))
X-Response-Timestamp: 1711234567891

Verify Response

expected = hmac_sha256(secret_key, response_body)
if signature != expected:
    raise SecurityError("Response tampered")
