⚡ Processing Pipelines

Document Pipeline

Upload → Sanitize → Extract → Store

PDF, DOCX, DOC, TXT | Max: 10MB

Image Pipeline

Upload → Sanitize → Compress → Store

WebP | Quality: 85%+ | Max: 1MB
